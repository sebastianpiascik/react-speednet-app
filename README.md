# Speednet App

### Treść zadania

```
1. Zadanie będzie polegało na wygenerowaniu listy z danych zamockowanych w kodzie
2. Każdy element listy będzie miał checkboxa (checkbox zaznaczony lub nie na podstawie pola is_unread)
3. Trzeba będzie obsłużyć zmianę checkboxa
4. Wyświetlić informacje: ile elementów na liście jest aktualnie zaznaczonych
5. Obsłużyć routing - przechodzenie z listy do detalu danego elementu i na odwrót
{
    id: 100795229,
    from: 'Kalkulator OC/AC - Onet <mailingi@onet.pl>',
    sent_date: '2021-12-13 09:00:03',
    is_unread: false,
    subject: 'Zaoszczędź! OC już od 230 zł',
    snippet: ''
}
6. Przykładowe dane mają tylko dwa elementy, ale równie dobrze mogłoby ich być 100, 1 000 lub 10 000
7. Wybór technologii: React, reszta wg. Twoich preferencji, używanie internetu jest możliwe a nawet wskazane.
```

## Development

```
npm install
npm start
```
